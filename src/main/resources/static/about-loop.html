About Loop

As a next-generation, ahead-of-the-curve and robust continuous-testing-enablement platform, LOOP is equipping many enterprises as they succeed to achieve seamless integration with CI/CD/DevOps ecosystems while arming their agile teams with a new found speed and flexibility.

Loop framework integrates with source code like Git, BitBucket etc. and planning tools like JIRA. It can also hook up to Continuous Integration tool like Jenkins for automated build and deployment. It can run the tests either on a distributed platform or on centralized server. It supports parallel test execution thereby reducing the feedback time to the development team. Loop also supports and recommends "shifting-left" performance tests and can help you execute and report tests designed with tools like JMeter. Loop is not only for UI or performance tests. It integrates API testing as the API tests are more reliable and faster than UI tests. It can integrate feed from monitoring tools like Splunk/ Nagios and helps you test continuously.
